Komodo Features

  • Advanced Editor

    Code folding, split view, multi-window editing, smart syntax checking and highlighting. Powerful snippets and macros, toolbox and more productivity-enhancing features.

  • Code Intelligence

    Auto-complete and calltips. Fast code navigation with Code Browser. IDE ONLY: Transform projects easily with Code Refactoring - rename class members, extract methods.

  • IDE ONLY

    Integrated Tools

    Version control integration for Subversion, Mercurial, Git, Perforce, Bazaar, CVS. Integrated unit testing. Database Explorer. Code Profiling for Python and PHP.

  • Clean User Interface

    Sleek UI frees up editing space. Undock and move side panes. Scroll code via minimap. Navigate previous files with Breadcrumbs. Sort and group open files.

  • IDE ONLY

    Team Capabilities

    Edit with multiple users simultaneously with Code Collaboration. Deploy and debug to a shared cloud environment with Stackato Platform-as-a-Service (PaaS).

  • Cross-Platform

    Identical features for Mac, Linux and Windows. User-based license: you can install on as many computers and platforms as you use.

  • IDE ONLY

    Graphical Debugging

    Debug PHP, Python, Perl, Ruby, Tcl, Node.js. Debug locally and remotely. Configure breakpoints. Create and debug regular expressions with Rx Toolkit.

  • Add-Ons and Customization

    Vi and emacs key bindings. Install Add-ons for specific features. Sync settings across multiple machines. Customize preferences and skins. Includes Abyss, new dark UI theme.
